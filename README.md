# My Favorite Movies
This Python code creates a pandas DataFrame with information about some of my favorite movies, and then displays the data as a table. The code uses the pandas library to create the DataFrame and the display() function to show the table.

Requirements
This code requires the following libraries:

pandas
You can install the required libraries using pip:

Copy code
pip install pandas

To use this code, simply run the Python script in a Jupyter Notebook or any Python environment that has the required libraries installed.

Example Output
yamlMy Favorite Movies
    Movie Title               Year Released      Director
0   The Shawshank Redemption  1994              Frank Darabont
1   The Godfather             1972              Francis Ford Coppola
2   The Dark Knight           2008              Christopher Nolan

Note that the exact output may vary depending on the environment and the size of the table.
